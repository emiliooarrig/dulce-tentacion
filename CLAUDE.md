Estas diseñando una pagina de repostería llamada Dulce Tentación.

La estructura general de la aplicación es:
- Backend: 
    - Lenguaje de programación: JavaScript
    - Framework: Express
    - Base de datos: MySQL
    - Almacenamiento de imagenes: Multer
- Frontend:
    - Lenguaje de programación: JavaScript
    - Framework: React
    - Tipografía: Montserrat y Tangerine

Las funcionalidades principales son:
- Login de usuarios para administración de pagina
- CRUD de productos
- CRUD de categorías
- CRUD de usuarios administradores

La paleta de colores es la siguiente, donde no debes de usar gradientes de color entre ellos :
- #FFC8DD
- #CDB4DB
- #FFAFCC
- #BDE0FE
- #A2D2FF

El diseño de la pagina debe ser moderno, limpio, elegante y responsivo, con las siguientes secciones:
- Inicio: 
    - Presentación de la pagina con un titulo grande que diga "Dulce Tentación" y un subtitulo que diga "Pastelería Artesanal"
    -  Descripción breve de la pagina
    - Productos destacados
    - Botón de "Ver productos"
- Productos: 
    - Lista de productos en forma de tarjetas con nombre, descripción, precio e imagen
    - Los productos se deben de mostrar de forma dinamica desde la base de datos
    - Filtros por categoría
- Contacto: 
    - Información de contacto
